Functional Javascript

The Functional Javascript library defines functions for using Javascript for functional programming (FP). It defines the standard FP functions such as map, reduce, select. It also defines higher-order functions (curry, rcurry, partial) for partial function application; and it accepts strings that define lambda expressions, such as 'x -> x+1' or 'x+1', as synonyms for the more verbose Javascript expressions such as function(x) {return x+1}.

Functional enables expressions such as these:

      map('x*x', [1,2,3,4])
        → [1, 4, 9, 16]
      some('x>2', [1,2,3,4])
        → true
      until('x>10', 'x*x')(2)
        → 16
      map(pluck('length'), ['two', 'words'])
        → [3,5]
      map(invoke('toUpperCase'), ['two', 'words'])
        → ["TWO", "WORDS"]
      map(compose(invoke('toString'), 'x*x'), [1,2,3,4])
        ["1", "4", "9", "16"]
      map(compose(pluck('length'), invoke('toString'), 'x*x'), [1,2,3,4])
        [1, 1, 1, 2]
      map('x -> (x*x).toString().length', [1,2,3,4])
        [1, 1, 1, 2]

Functional doesn't define as many collection functions as a library such as Prototype. It can, however, be used in conjunction with such a library. The code that generates this page is an example of this.
