Functional Javascript

The Functional Javascript library defines functions for using Javascript for functional programming (FP). It defines the standard FP functions such as map, reduce, select. It also defines higher-order functions (curry, rcurry, partial) for partial function application; and it accepts strings that define lambda expressions, such as 'x -> x+1' or 'x+1', as synonyms for the more verbose Javascript expression function(x) {return x+1}.

Functional doesn't define as many collection functions as a library such as Prototype. It can, however, be used in conjunction with such a library. The code that generates the documentation and examples, below, does this.

If you can read this, you either have JavaScript turned off or you're using a browser I haven't tested. Either way, you're missing out on the page content, which is generated from the source code. Try Mozilla Firefox or Apple Safari.
